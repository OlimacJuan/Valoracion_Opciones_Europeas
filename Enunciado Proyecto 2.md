Usted debe realizar un programa que le permita al usuario cambiar estos parámetros y ejecutar la simulación para obtener otros posibles resultados.
2. 	1. Plantee las ecuaciones de Black-Scholes que rigen el modelo estocástico para $C$.
	2. Proponga las condiciones de frontera para $C$.
	3. Transforme la ecuación de Black-Scholes para $C$ en una ecuación de difusión y resuélvala analíticamente si es posible.
	4. Dibuje claramente la superficie de la función $C = C(S,t)$ para diferentes valores de la tasa de interés y la volatilidad (sigma).
